Текстурирование моделей
Есть набор 3D моделей (торс, руки, ноги, ступни) в формате Wavefront OBJ. Так же имеется набор текстур для этих моделей.
Требуется — используя любые комбинации из элементов нарисовать для них текстуры одежды по картинкам (скорее всего одежда героев комиксов Dynamite Enteriment).
Таким образом, можно комбинировать любое сочетание торса, ног обуви для получения наилучшего эффекта. По две модели каждого элемента и «референсные» текстуры для них в приложении. Так же прикладывается картинка желаемой текстуры. 
Правила наименования моделей и текстур
xxxxx_mNNN_g — кисти рук 
xxxxx_mNNN_u — торс 
xxxxx_mNNN_l — ноги 
xxxxx_mNNN_f — ступни
Цвет кожи с базовых моделей править НЕ НАДО. Если на картинке какие-то элементы не видны, то на усмотрение исполнителя.
Если какой-то элемент (воротник и плащ в костюме Вампиреллы например) невозможно нарисовать для набора моделей, то этот элемент можно отбросить.

Общее количество «комплектов» будет 5-6.
