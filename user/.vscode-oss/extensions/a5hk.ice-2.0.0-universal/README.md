# [Ice](https://marketplace.visualstudio.com/items?itemName=a5hk.ice)

Also available for [vim](https://github.com/a5hk/ice/blob/eeb01fa8fc6a50412a7f68a3193370ecbbf3ca77/vim/colors/), [bat](https://github.com/a5hk/ice/blob/eeb01fa8fc6a50412a7f68a3193370ecbbf3ca77/bat/), and [Windows Terminal](https://github.com/a5hk/ice/blob/eeb01fa8fc6a50412a7f68a3193370ecbbf3ca77/windows-terminal/).
